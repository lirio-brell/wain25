Formatting Instructions

This is an open call for papers. We invite both full papers (max. 8 pages) that describe mature work and short papers (4-6 pages) that describe work-in-progress or case studies. Only original and high-quality papers formatted using the IEEE 2-column format (Latex Template), including the bibliography and any possible appendices, will be considered for review.
